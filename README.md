## Nécessité

#### _Afin de faire fonctionner notre programme, nous avons besoin de :_

- **NodeJs**
- **Google Chrome ou/et Mozilla Firefox**

#### _Installation du projet :_

- **git clone {projet}**
- **cd {projet}/pokejs**
- **npm i**

#### _Lancement du projet :_

- **npm start**
